### An Easy-To-Use Grocery Shopping List & Tracker App with Core Data & SwiftUI

## Features
### Home
- Simply create, edit, and delete grocery items to keep track of your grocery items
- Cleary filter items by categories, search query or both
- Sort grocery items by date or amount in descending or ascending order

### Grocery Shopping List
- Easily add grocery shopping items which need buying
- Keep record for grocery shopping list

### Settings
- Support mutiple theme color options
- Compatible with dark mode
- Provide English and Chinese localization

### Tech Skills
- Support iPad layout
- Purely **`SwiftUI`**
- **`Core Data`**
- **`MVVM`** architecture
